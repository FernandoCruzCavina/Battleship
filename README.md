# Battleship

This game was developed to apply matrix and equation concepts. Of course, the project was supervised by my professors, Maricélia and Andrei. 

Someone ask how this game works ? Oh let me explain! 

You play on the green grid and try to find the enemy ships' positions until you destroy all of them. Meanwhile, your opponent tries to do the same. The player who destroys all enemy ships first wins!
It's a classic battleship game, but it has a new feature: the Radar. Once you press this button, the system alerts you to the position of the enemy ship.

You really need to know the x and y coordinates to understand the position for win the game!🚢💥

## How to Play ?

You only need Python 3.1 or later installed.

Then install the required library Pygame:

```sh
pip install -r requirements.txt
```

You ready now, enjoy the game!
```sh
python main.py
```
